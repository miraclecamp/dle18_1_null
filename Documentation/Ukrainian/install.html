Документація доступна у файлі readme.chm.

1. Установлення скрипта на сервер

Установлення скрипта на сервер відбувається в автоматичному режимі і не має викликати особливих проблем, але Вам потрібно виконати деякі дії з налаштування прав доступу на теки.

  1. Для початку скопіюйте всі файли з теки upload у цьому архіві на Ваш сервер.
  2. Установіть для теки templates і всіх вкладених у неї тек права на запис (CHMOD 777)
  3. Установіть права для запису (CHMOD 777) для тек backup, uploads, а також для всіх тек, що містяться всередині них.
  4. Установіть права для запису (CHMOD 777) для тек /engine/data/, /engine/cache/.
  5. Запустіть файл install.php і дотримуйтеся його інструкцій. Автоматичний установлювач перевірить усі потрібні файли і налаштує під'єднання до бази даних. Після завершення встановлення Ви можете переглянути роботу скрипта на демонстраційній сторінці http://website.com/index.php. Для доступу до адмінпанелі потрібно запустити http://website.com/admin.php

Увага! Після встановлення або оновлення скрипта, на сервері обов'язково видаліть файл install.php, інакше буде можливо знову ініціювати процес установлення скрипта і цим знищити поточну структуру даних.

Надіємося, що робота з нашим скриптом потішить Вас.
