Script installation on the server

Script installation on the server is fully automated and should not cause any problems, but you need to perform some steps to set access rights to folders.

  1. First, copy all the files from the upload folder of this archive to your server.
  2. Make the templates folder and all its subfolders writable. (CHMOD 777)
  3. Make backup, uploads folders writable, as well as all the folders inside them. (CHMOD 777)
  4. Make the following folders writable (CHMOD 777) /engine/data/, /engine/cache/.
  5. Run the install.php file and follow the instructions. Automatic installer will check all the necessary files and adjust connection to the database. After installation is complete, you can go to demo page http://website.com/index.php and see how the script works. To access the administration panel, you must go to http://website.com/admin.php

Attention: When you install or upgrade the script on the server, be sure to remove the install.php file. Otherwise it will be possible to re-initiate the script installation process, thus destroying the existing data structure.
