Brief information on upgrade

Dear friends!

Thank you for choosing our script to be used for your website. We are making great efforts to make the script accessible and understandable as much as possible, but we strongly recommend you to read the full documentation of the script before you start working with it. Documentation is available in the readme.chm file.

Script update

  1. Make a backup copy of the database and all the script files on the server.
  2. Update all files on the server from the /upload/ folder of this archive (except /templates/)
  3. Log in your adminpanel http://yoursite.com/admin.php, and follow the instructions.
  4. Make the required changes to your templates, according to the list of changes posted at: https://dle-news.com/templates-changelog.html

Attention: When you install or upgrade the script on the server, be sure to remove the install.php file. Otherwise it will be possible to re-initiate the script installation process, thus destroying the existing data structure.

We hope that that you enjoy working with our script.
